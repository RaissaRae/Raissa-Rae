Mousetrap

This was a first year class called IoT Essentials, we had to make a mousetrap using different components.

Overview

Our permanent evaluation was to create a mousetrap using the components that we learnt about. The components were a Raspberry Pi, a stepper motor, an ultrasonic measurement, LCD and we also made used of uBeac. Despite never taking Physics in high school, I really enjoyed this class and the project. The stepper motor was used to close the door once the mouse was detected, the ultrasonic measurement component was used to show how far the mouse was, we used the LCD to display the amount of the times the trap was closed, the current date and time.
