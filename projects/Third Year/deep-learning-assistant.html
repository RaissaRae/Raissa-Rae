Personal Learning Assistant

This was a third year semester 1 for our Deep Learning class.

Overview

The Personalized Learning Assistant is an AI-driven educational tool that helps users engage with course materials more effectively. It provides tailored summaries, detailed explanations, and quizzes based on uploaded course documents. The application is built using Streamlit and employs RAG (Retrieval-Augmented Generation), a vector database, and Groq API (default LLM) to deliver relevant and personalized learning experiences. This was a group project with two other students, we all did the basics which was using Streamlit for the interface, the content and question answering agent. I worked on the Speech to Text section, Text to Speech section.
