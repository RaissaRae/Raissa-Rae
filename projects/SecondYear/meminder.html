Meminder

This was a second year course called Project PHP.

Overview

This was a course that worked hand in hand with Web Application Development with PHP and Requirements Analysis. We were given a report which had been complied by another group, and we had to make appropriate changes after speaking to the client. This time, the client was Laurien Stroobants. We had to create an application for people with memory problems, who have an informal carer who would provide support to the primary user. We had to integrate different users other than the primary user (patient), we had to have carers and admins in order for the app to be running.I was document lead so besides handling all the paperwork, I had to create sample data for the database, I had to create the profile page for each user, and the guest comment page for the guests who had could add messages for the primary user.
